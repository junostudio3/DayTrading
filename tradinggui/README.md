# PulseTrade Dashboard

React and TypeScript dashboard scaffolded with Vite for a day trading workflow.

## Included UI

- Session overview with liquidity, P&L, buying power, and win-rate cards
- Intraday price pulse panel with a stylized chart surface
- Momentum watchlist, order book, open positions, execution notes, and sector heat
- Responsive single-page layout tuned for desktop and tablet use

## Scripts

- `npm run dev` starts the Vite development server
- `npm run build` performs the TypeScript build and bundles the app for production
- `npm run lint` runs ESLint across the project

## Notes

- The dashboard currently uses static market data for layout and interaction prototyping.
- The next integration step is wiring the view to a live or simulated market data source.
